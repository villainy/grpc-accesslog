Features
--------

* Write stdout logs for every RPC request
* Log messages built with customizable callback handlers


Requirements
------------

* Python 3.6+
* grpc-interceptors 0.13+


Installation
------------

You can install *gRPC Access Log* via pip_ from PyPI_:

.. code:: console

   $ pip install grpc-accesslog
